# Semantic Search App

This is a semantic search application that allows you to search for documents and get a summary of the top results. It uses a Flask frontend, a FastAPI backend, and Qdrant as the vector database.

## Setup Instructions

### Prerequisites

- Python 3.7 or higher
- Docker and Docker Compose
- OpenAI API key (if using OpenAI for summarization)
